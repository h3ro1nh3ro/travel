# KickOnion

Мобильное приложение для скейтеров и уличных спортсменов Санкт-Петербурга. Челленджи с AI-оценкой трюков, онлайн-игра S.K.A.T.E., карта парков и споты города.

---

## Что это

KickOnion объединяет скейтеров, BMX-еров, роллеров и других уличных спортсменов вокруг соревновательного контента. Пользователь снимает трюк, получает AI-оценку, соревнуется в онлайн S.K.A.T.E. с другими игроками и находит парки на карте города.

---

## Разделы

### ИГРАТЬ
**S.K.A.T.E.** — классическая игра один на один (или группа). Один задаёт трюк, остальные повторяют. Кто не повторил — получает букву. Собрал S-K-A-T-E — выбыл. Игра идёт в реальном времени через комнаты.

**ЧЕЛЛЕНДЖ** — конкретный трюк с описанием и сложностью. Снимаешь видео, AI анализирует технику и выставляет оценку 0–100. Очки идут в общий рейтинг сезона.

### НОВОСТИ
Ивенты, контесты, апдейты приложения, скейт-культура Петербурга.

### АУКЦИОН
P2P-продажа снаряжения между пользователями без посредников.

### ПАРКИ
Интерактивная карта скейт-парков, споты и трассы СПб. Фильтр по виду спорта, карточки с режимом работы, покрытием, отзывами и рейтингом.

### ЛИДЕРБОРД
Общий рейтинг по очкам сезона. Топ по каждому виду спорта.

## Темизация

Каждый вид спорта имеет свою цветовую тему. Тема хранится в `localStorage` и применяется через CSS-переменные (`--t-bg`, `--t-card`, `--t-primary` и др.) без мигания при загрузке (inline-скрипт в `<head>`).

| Спорт | Primary | Фон |
|-------|---------|-----|
| Skate | `#a8956e` (золото) | `#0a0a0a` |
| BMX | `#00BBFF` (синий) | `#05080f` |
| Ролики | `#6dba4a` (зелёный) | `#eeede8` |
| Самокат | `#FF6B35` (оранжевый) | `#0e0b07` |
| Серфскейт | `#00A89D` (бирюза) | `#f4ede0` |
| Бег | `#7d8fa0` (серо-синий) | `#181c20` |

## Запуск

```bash
# Установка зависимостей (из корня монорепо)
npm install

# Дев-сервер фронтенда
cd frontend && npm run dev
# → http://localhost:3000

# Бэкенд
cd backend && npm run dev
# → http://localhost:3001
```

### Сборка APK

```bash
cd frontend
npm run generate        # SSG-сборка
npx cap sync android    # Синхронизация с Android-проектом
npx cap open android    # Открыть в Android Studio → Build APK
```

## Ключевые решения

**Capacitor WebView** — `position: fixed` ненадёжен в WebView. Используем flex-column layout (`height: 100dvh; overflow: hidden`) с `flex-1 overflow-y-auto` для основного контента и `flex-shrink-0` для хедера и футера.

**Навигация** — `pageTransition: false` устраняет чёрный экран между переходами. В Skate-игре используем `router.push()` вместо `NuxtLink` для надёжной навигации.

**Тема без мигания** — inline-скрипт в `<head>` восстанавливает CSS-переменные из localStorage до первого рендера.

**AI-оценка** — пока работает на mock-данных. Планируется подключение Claude API для реального анализа видео.
